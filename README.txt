Site web permettant d'avoir des informations complètes d'un champion de League of Legends.

Le site utilise l'API REST pour récuperer des informations via l'id d'un champion. 

La route : '/' nous permet d'avoir les infos du champion

La route : '/SkinPerso' nous permet d'avoir les differents skins du perso grace à son id et l'id de son skin
